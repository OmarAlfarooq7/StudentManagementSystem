# StudentInfoSystem - C# Windows Forms Application

نظام لإدارة معلومات الطلاب باستخدام C# و Windows Forms، مع قاعدة بيانات SQL Server.

## الميزات
- إضافة وتعديل وحذف الطلاب والدورات والأقسام.
- تسجيل دخول المستخدمين مع صلاحيات.
- قاعدة بيانات SQL Server مع مستخدم افتراضي ""admin"".

## ملفات المشروع
- FormStudent.cs : نموذج إدارة الطلاب.
- Student.cs : تعريف كلاس الطالب.
- FormStudentInfo.cs : النموذج الرئيسي لعرض الطلاب.
- DbStudent.cs : عمليات قاعدة البيانات.
- Database/student.sql : سكربت إنشاء قاعدة البيانات والجداول مع مستخدم افتراضي.
- .gitignore : تجاهل الملفات المؤقتة.
